To create **implementation-ready protocols** for ADR-035, we need to extract:

### For EACH PROTOCOL:

1. **Metadata**
   - Varietal name
   - Color (RED/WHITE/ROS√â)
   - Expected duration (days)
   - Winery/Source
   - Vintage(s) applicable

2. **Timeline Steps** (in order)
   - Day N: Step description
   - Expected timing (hour window)
   - Observation targets (what to check?)
   - Decision points (if X, then Y)

3. **Environmental Parameters**
   - Temperature range (min-max)
   - H2S monitoring frequency
   - Punch-down frequency (if applicable)
   - Settling/racking schedule

4. **Critical Thresholds**
   - BRIX targets (start, 1/3, 2/3, dry)
   - pH range
   - Temperature limits (abort points?)
   - H2S severity (when to escalate?)

5. **Varietal-Specific Notes**
   - Unique challenges (e.g., "Pinot sensitive to H2S")
   - Opportunities (e.g., "Petite Sirah benefits from warm fermentation")
   - Decision trees (e.g., "If H2S detected, add aeration")
